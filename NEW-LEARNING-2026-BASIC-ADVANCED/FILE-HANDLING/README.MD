## CONCEITOS BÁSICOS

- COBOL TRABALHA COM 3 TIPOS DE ARQUIVOS

----

SEQUENCIAL:

LEITURA LINHA POR LINHA - DO INICIO AO FIM

EX: LOGS E RELATÓRIOS

----


INDEXADO: 

ACESSO DIRETO POR CHAVE - ESTILO BANCO DE DADOS

EX: CADASTROS


----


RELATIVO:

ACESSO POR NÚMERO DE REGISTRO

EX: É RARO HOJE EM DIA
